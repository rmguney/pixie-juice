# Pixie Juice

**Security focused and high performance media processing engine in Rust and C, for WebAssembly targets.**

Pixie Juice is a client side WebAssembly application built with Rust and C that optimizes images and 3D models without uploading files to any server. Your data never leaves your device, ensuring complete privacy while delivering fast media processing.

## Features

### Image Processing

- **Formats**: JPEG, PNG, WebP, GIF, BMP, TIFF, SVG, ICO, PDF, AVIF, HEIC
- **Optimization**: Intelligent compression with quality controls
- **Conversion**: Convert between any supported formats
- **Real-time**: Instant preview and processing

### 3D Model Processing

- **Formats**: OBJ, GLTF, GLB, PLY, STL, FBX, DAE, USDZ, 3DS
- **Optimization**: Mesh simplification and texture optimization
- **Conversion**: Convert between 3D formats
- **Preview**: Interactive 3D model viewer

### Performance

- **WebAssembly**: Rust + C hybrid for maximum speed
- **Client-side**: No server uploads, complete privacy
- **Batch processing**: Handle multiple files simultaneously
- **Modern UI**: Next.js 15 + React 19 interface

## Getting Started

1. **Visit**: [pixiejuice.vercel.app](https://pixiejuice.vercel.app/)
2. **Upload**: Drag and drop your images or 3D models
3. **Optimize**: Adjust quality settings and preview results
4. **Download**: Get your optimized files instantly
